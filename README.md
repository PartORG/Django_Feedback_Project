# Django_Feedback_Project

A simple example of a Django project to demonstrate basic web application development.

## Requirements

- Python 3.10
- Django 4.0

## Installation

To set up the project, follow these steps:

1. Clone the repository:
   ```sh
   git clone https://github.com/PartORG/Django_Feedback_Project.git
   cd Django_Feedback_Project
   ```

2. Create a virtual environment and activate it:
   ```sh
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. Install the dependencies:
   ```sh
   pip install Django==4.0
   ```

## Usage

To run the project, execute the following command:

```sh
python manage.py runserver
```

This will start a development server on `http://127.0.0.1:8000/`. You can access your Django application by navigating to this URL in your web browser.

For more information on how to use and extend the project, refer to the [Django documentation](https://docs.djangoproject.com/en/stable/).
